# 教室信息管理系统 java378

## 项目概述

本项目是一款基于SSM（Spring + SpringMvc+ Mybatis）和Vue的前后端分离的教室信息管理系统。采用MVC三层体系结构，实现了高效的系统管理。

## 技术栈

- 前端：Vue.js
- 后端：Spring、SpringMvc、Mybatis
- 数据库：MySQL
- 项目构建：Maven

## 功能模块

- 教室信息管理
- 课程安排管理
- 用户权限管理

## 系统角色

- 管理员
- 教师
- 学生

## 运行环境

- 操作系统：Windows/Linux/MacOS
- 数据库：MySQL 5.7+
- JDK版本：1.8+
- IDE：IntelliJ IDEA/VSCode

## 部署步骤

1. 安装MySQL数据库
2. 导入项目SQL文件
3. 使用Maven构建项目
4. 部署前端Vue项目
5. 启动后端服务

## 目录结构

```
教室信息管理系统
|-- frontend（前端项目目录）
|-- backend（后端项目目录）
    |-- src
        |-- main
            |-- java
            |-- resources
            |-- webapp
    |-- pom.xml
```

## 核心亮点

- 前后端分离，便于维护和扩展
- 界面简洁，操作简单
- 采用MVC设计模式，层次清晰
- 数据库采用MySQL，稳定可靠
